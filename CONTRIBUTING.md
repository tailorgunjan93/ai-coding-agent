# Contributing to AI Coding Agent

Thank you for considering contributing to the AI Coding Agent! We appreciate your interest and want to make the contribution process as smooth as possible. Below are the guidelines to follow.

## Setup
1. **Fork the Repository**: Start by forking the repository to your own account.  
2. **Clone Your Fork**: Clone the forked repository to your local machine using:
   ```bash
   git clone https://github.com/<your-username>/ai-coding-agent.git
   ```
3. **Install Dependencies**: Navigate to the project directory and install the required dependencies. This usually requires running:
   ```bash
   npm install
   ```
   Or the equivalent command for your setup.

- Ensure that all tests pass before submitting a Pull Request (PR).

## Code Style
- Follow the coding conventions outlined in the project documentation (link to any style guides, e.g., Airbnb, Google).  
- Make sure to lint your code before making a PR. Typically, this can be done by running:
  ```bash
  npm run lint
  ```

## Pull Request (PR) Process
1. **Create a Feature Branch**: Before making changes, create a new branch for your feature/fix:
   ```bash
   git checkout -b feature/my-new-feature
   ```
2. **Make Your Changes**: Implement your feature or fix the bug.
3. **Commit Your Changes**: Ensure your commit message is clear; it should describe what you’ve changed. 
4. **Push to Your Branch**: Push your changes back to your fork:
   ```bash
   git push origin feature/my-new-feature
   ```
5. **Create a Pull Request**: Go to the original repository and select “New Pull Request.” 
6. **Respond to Review Comments**: Be prepared to make changes based on feedback from the maintainer(s).

We look forward to your contributions!
